SICP as PDF File

This is a repository to archive a PDF copy of the famous computer-science textbook Structure and Interpretation of Computer Programs, Second Edition by Harold Abelson, Gerald Jay Sussman, and Julie Sussman.

This PDF copy has been created from github.com/sarabander/sicp-pdf, a project by Andres Raba to render the textbook with Texinfo and LaTeX. This PDF copy is versioned as: Unofficial Texinfo Format 2.andresraba5.6.
